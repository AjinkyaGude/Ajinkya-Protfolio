    Project's

    DevOps - Website Deployemnt using latest technologies

    Technologies Used

    • Application Server: Apache Tomcat for hosting the web application.
    • Message Broker: RabbitMQ for efficient asynchronous messaging.
    • Database Management: MySQL for reliable data storage and retrieval.
    • Caching Layer: Memcached to optimize performance and reduce database load.
    • Reverse Proxy & Load Balancing: Nginx for handling HTTP requests and distributing traffic.
    • Virtual Machines & OS: CentOS and Ubuntu for running services in a virtualized

    Brief Summary

    This project automates the deployment and scaling of a web application using Apache Tomcat, RabbitMQ, MySQL, and Memcached for optimized performance. Nginx acts as a reverse proxy and load balancer, while virtual machines running CentOS and Ubuntu provide a stable infrastructure. RabbitMQ enables efficient asynchronous messaging, and monitoring/logging solutions ensure system reliability. This project highlights my expertise in DevOps automation, infrastructure management, and scalability.
